## 🎯 오늘 배울 내용

### Session 0: 환경 세팅
- VS Code와 Copilot 설치
- 첫 프롬프트 작성

### Session 1: 회의록 자동 정리
- 회의록에서 Action Item 자동 추출
- 구조화된 회의록 생성

### Session 2: 문서 자동 작성
- PRD(Product Requirement Document) 5분 만에 작성
- 다양한 문체로 자동 변환

### Session 3: UX 리서치 분석
- 사용자 인터뷰 데이터 자동 분석
- Python으로 인사이트 추출

### Session 4: 데이터 분석
- CSV 파일 자동 분석
- 차트와 그래프 자동 생성

### Session 5: 웹 애플리케이션 만들기
- Flask로 간단한 웹 서버 구축
- 할 일 관리 앱 제작
- 브라우저에서 바로 실행

### Session 6: 업무 자동화
- 반복 작업 스크립트 작성
- 이메일 자동 추출 등

### Session 7: 워크플로우 통합
- 모든 기술을 하나로 연결
- 나만의 자동화 워크플로우 설계
