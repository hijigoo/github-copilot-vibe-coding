# GitHub Copilot 워크숍

> **코딩 경험이 없어도 괜찮습니다!**  
> AI와 함께 업무를 자동화하고 생산성을 높여보세요.

---

## 🎯 누구를 위한 워크숍인가요?

- **PO/기획자**: 기획서와 PRD를 빠르게 작성하고 싶은 분
- **UX/UI 디자이너**: 사용자 리서치 데이터를 효율적으로 분석하고 싶은 분
- **마케터**: 데이터 분석과 보고서 작성을 자동화하고 싶은 분
- **모든 비개발자**: 반복 작업을 자동화하고 시간을 절약하고 싶은 분

**코딩 경험 불필요** | **강사와 함께** | **바로 활용 가능**

---

## ⏱️ 얼마나 절약할 수 있나요?

### 실제 시간 절감 효과
- 📝 **회의록 정리**: 30분 → 5분 (83% 절약)
- 📄 **PRD 문서 작성**: 2시간 → 5분 (96% 절약)
- 📊 **데이터 분석**: 5시간 → 20분 (93% 절약)
- 🌐 **웹 페이지 제작**: 2일 → 30분 (99% 절약)

**→ 주당 평균 17시간 이상 절약 가능!**

## 🚀 시작하기

### 1️⃣ 설치하기
**Session 0**에서 필요한 모든 것을 설치합니다:
- VS Code 에디터 (강력한 텍스트 편집기)
- GitHub Copilot 확장 (AI 도우미)
- Python 환경 (데이터 분석 도구)

**💡 Tip**: 설치가 처음이라 걱정되시나요? 강사가 함께 진행하니 걱정 마세요!

👉 **[Session 0: 환경 세팅 바로가기](<sessions/0. 환경 세팅.md>)**

### 2️⃣ 빠르게 시작하기
설치 후 바로 사용법을 배웁니다:

- 첫 프롬프트 작성하기
- 실습 준비 완료!

👉 **[Session 1: 빠른 시작 바로가기](<sessions/1. Copilot 첫 사용하기.md>)**

- 모든 샘플 데이터 제공

## ❓ 자주 묻는 질문

### Q: 코딩을 전혀 모르는데 괜찮나요?
**A**: 네! 이 워크숍은 비개발자를 위해 만들어졌습니다. 모든 코드는 복사 & 붙여넣기로 진행됩니다.

### Q: Python을 배워야 하나요?
**A**: 아니요! Python은 도구일 뿐입니다. Copilot이 코드를 작성해주므로 문법을 몰라도 됩니다.

### Q: 실습 중 막히면 어떻게 하나요?
**A**: 각 세션 문서에 "자주 묻는 질문"과 "트러블슈팅" 섹션이 있습니다. 그래도 안 되면 강사에게 질문하세요!

### Q: 회사에서 바로 사용할 수 있나요?
**A**: 네! 모든 예제는 실무에서 바로 적용 가능하도록 설계되었습니다.
