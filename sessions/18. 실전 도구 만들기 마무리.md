# 4. 실전 도구 만들기 마무리

## 🎉 완료!

Copilot과 함께 현업 전문가도 바로 배포 가능한 웹앱 두 개를 완성했습니다!
요구사항만 명확히 말해도 CRUD, 계산 로직, 저장 기능까지 포함된 실전 도구를 몇 분 안에 만들 수 있다는 것을 확인했습니다.

---

## 📌 핵심 포인트

**2. 제품 아이디어 관리 도구**
- 아이디어/피드백/요구사항을 한 화면에서 입력·분류·필터링
- 카테고리별 분류 (아이디어/피드백/요구사항), 우선순위 색상 구분 (높음=빨강, 중간=주황, 낮음=초록)
- TXT(JSON) 저장/불러오기, 수정/삭제 기능, 카드 UI를 Copilot 프롬프트만으로 구현
- 반응형 디자인으로 모바일에서도 사용 가능

**3. 예약·수수료 계산기**
- 1박 요금, 숙박일수, 쿠폰, 플랫폼 수수료(%)를 입력하면 즉시 계산
- 고객 최종 결제 금액 · 플랫폼 수수료 금액 · 업주 수령액을 한눈에 표시
- 쿠폰 과다 적용 시 경고 토스트, 결과 클립보드 복사 버튼
- 예시 시나리오 2개를 클릭 한 번으로 자동 입력

| `revenue_calculator.html` | 요금 계산 로직, 예시 시나리오 자동 입력, 결과 복사 기능이 있는 정산 계산기 |

## 🧪 프롬프트 패턴

### 패턴 1: 도구 전체 요청
```
[사용자/팀]을 위한 [도구명].html을 만들어줘.
기능: [입력 필드, 버튼, 동작을 상세히 나열]
디자인: [색상, 레이아웃, 반응형 요구사항]
```

### 패턴 2: 저장/불러오기 기능 추가
```
데이터를 TXT(JSON)로 저장/불러오기 기능을 추가해줘.
저장 버튼과 불러오기 버튼을 만들고,
파일 형식은 JSON으로 해줘.
```

### 패턴 3: 테스트 시나리오 자동화
```
예시 시나리오를 미리 채워둔 버튼을 추가하고,
클릭 시 입력값이 자동으로 기입되게 해줘.
```

### 패턴 4: 계산 로직 검증
```
[계산식을 명확히 설명]
잘못된 입력(예: 쿠폰이 총액보다 큼)은 경고 메시지를 띄워줘.
결과를 클립보드에 복사하는 버튼도 추가해줘.
```

💡 **핵심**: 기능·디자인·파일명을 한 번에 명시하면, Copilot이 즉시 실행 가능한 단일 HTML 파일을 만들어줍니다!
