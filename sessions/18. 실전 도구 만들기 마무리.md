# 15. 실전 도구 만들기 마무리

## 🎉 완료!

Copilot과 함께 현업 전문가도 바로 배포 가능한 웹앱 두 개(아이디어 관리, 예약·수수료 계산기)를 완성했습니다. 요구사항만 명확히 말해도 CRUD, 계산 로직, 저장 기능까지 포함된 실전 도구를 몇 분 안에 만들 수 있다는 것을 확인했습니다.

---

## 📌 핵심 포인트

**Session 13 · 제품 아이디어 관리 도구**
- 아이디어/피드백/요구사항을 한 화면에서 입력·분류·필터링
- TXT(JSON) 저장/불러오기, 우선순위 색상, 카드 UI를 Copilot 프롬프트만으로 구현

**Session 14 · 예약·수수료 계산기**
- 숙박 요금, 쿠폰, 수수료를 입력하면 즉시 고객 결제 금액·플랫폼 수수료·업주 수령액 계산
- 예시 시나리오, 경고 토스트, 클립보드 복사 버튼 등 실무 편의 기능까지 포함된 단일 HTML 앱

---

## 📁 확보된 산출물
- `product_manager.html`: 입력 폼, 카테고리/우선순위 필터, TXT 저장/불러오기를 갖춘 아이디어 관리 툴
- `revenue_calculator.html`: 요금 계산 로직, 예시 시나리오 자동 입력, 결과 복사 기능이 있는 정산 계산기
- (선택) `q4_ideas.txt` 등 Copilot이 생성한 샘플 데이터를 불러오며 재사용할 수 있는 자료

---

## 🧪 프롬프트 패턴
```
1) "[사용자/팀]이 사용할 [도구명].html을 만들고, 아래 입력 필드와 동작을 모두 포함해줘"
2) "데이터를 TXT(JSON)로 저장/불러오기 기능을 추가하고, 버튼/토스트/필터 UI를 넣어줘"
3) "테스트 시나리오를 미리 채워둔 버튼을 추가하고, 클릭 시 입력값이 자동으로 기입되게 해줘"
```
기능·디자인·파일명을 한 번에 명시하면, Copilot이 즉시 실행 가능한 단일 HTML 파일을 만들어줍니다.
