### Step 3: 파일의 내용은 다음과 같습니다.

| 파일명 | 용도 | 내용 |
|--------|------|------|
| `sample_meeting_notes.txt` | 회의록 자동 정리 실습 | 실제 회의록 형식의 텍스트 (6개 회의, 17개 Action Items) |
| `user_feedback_data.txt` | 사용자 피드백 분석 실습 | 30명의 사용자 피드백 (연령대, 직업, 의견 포함) |
| `customer_satisfaction_survey.csv` | CSV 데이터 분석 및 리포트 생성 | 고객 만족도 설문조사 결과 (50건) |
| `customer_list_full.csv` | 업무 자동화 실습 | 고객 정보 및 예약 상태 (20건) |
| `email_extraction_sample.txt` | 정규표현식 실습 | 이메일 주소가 포함된 텍스트 |
| `feature_concept.txt` | PRD/문서 자동 작성 실습 | "스마트 가격 알림" 기능 기획 배경, 핵심 기능, 목표 지표 |
