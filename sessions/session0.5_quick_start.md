## Step 1: Copilot Chat 열기 (1분)

### 방법 1: 키보드 단축키
- **macOS**: `⌘ + I`
- **Windows**: `Ctrl + I`

### 방법 2: 사이드바
- 왼쪽 사이드바에서 **채팅 아이콘** 클릭
- 또는 상단 메뉴 → View → Chat

### ✅ 확인
Chat 창이 열리면 "Ask Copilot a question..." 입력창이 보여야 합니다.

---

## Step 2: 첫 프롬프트 작성하기 (3분)

### 실습 1: 간단한 인사

**프롬프트**:
```
안녕하세요! 저는 비개발자입니다. GitHub Copilot이 무엇을 도와줄 수 있나요?
```

**기대 결과**:
- Copilot이 자신의 기능을 설명합니다
- 비개발자가 활용할 수 있는 방법을 제안합니다

### 실습 2: 간단한 문서 생성

**프롬프트**:
```
회의 참석자 5명의 이름과 역할을 표로 만들어줘. 
참석자는 김철수(PO), 이영희(UX), 박지민(마케터), 최민수(개발자), 정수진(디자이너)야.
```

**기대 결과**:
Copilot이 다음과 같은 표를 생성합니다:

| 이름 | 역할 |
|------|------|
| 김철수 | PO |
| 이영희 | UX |
| 박지민 | 마케터 |
| 최민수 | 개발자 |
| 정수진 | 디자이너 |

---

## Step 3: 프롬프트 작성 팁 이해하기 (3분)

### ✅ 좋은 프롬프트 예시

**구체적으로**:
```
❌ "보고서 만들어줘"
✅ "지난주 프로젝트 진행 상황 보고서를 만들어줘. 
    완료된 작업, 진행 중인 작업, 다음 주 계획으로 구성해줘."
```

**형식 지정**:
```
❌ "데이터 정리해줘"
✅ "이 데이터를 표 형식으로 정리하고, 
    각 항목에 번호를 붙이고, 
    중요한 항목은 굵은 글씨로 표시해줘."
```

**예시 제공**:
```
❌ "이메일 작성해줘"
✅ "고객에게 제품 업데이트를 알리는 이메일을 작성해줘. 
    예: '안녕하세요, 이번 주 새로운 기능이 추가되었습니다...' 
    같은 친근한 톤으로."
```

## Step 4: 프로젝트 폴더 구조 파악하기 (3분)

### 세션 폴더 열기

1. VS Code에서 **Explorer** 열기 (`⌘+Shift+E` 또는 `Ctrl+Shift+E`)
2. `sessions` 폴더 찾기
3. 다음 파일들이 보이는지 확인:

```
sessions/
├── session1_meeting_notes.md      # 회의록 자동 정리
├── session2_document_automation.md # 문서 자동 작성
├── session3_ux_research.md        # UX 리서치 분석
├── session4_csv_report.md         # 데이터 분석
├── session5_web_app.md            # 웹 앱 만들기
├── session6_automation.md         # 업무 자동화
├── session7_workflow.md           # 워크플로우 통합
└── data/                          # 실습 데이터
```

## 🎓 핵심 개념 정리

### Copilot Chat이란?
- AI 페어 프로그래밍 도우미
- 자연어로 대화하며 작업
- 코드부터 문서까지 생성 가능

### 프롬프트란?
- Copilot에게 주는 지시사항
- 명확할수록 더 좋은 결과
- 여러 번 시도하며 개선 가능

### 비개발자도 사용할 수 있나요?
**네!** 이 워크숍의 모든 예제는:
- 코딩 경험 불필요
- 복사 & 붙여넣기로 진행
- 단계별로 자세히 설명

## ❓ 자주 묻는 질문

### Q: Copilot이 응답하지 않아요
**A**: 다음을 확인하세요:
1. 오른쪽 하단 Copilot 아이콘이 초록색인지 확인
2. 인터넷 연결 확인
3. VS Code 재시작
4. GitHub 계정 로그인 상태 확인

### Q: 프롬프트를 잘못 입력했어요
**A**: 괜찮습니다!
- 다시 입력하면 됩니다
- "다시 해줘" 또는 "이전 답변을 수정해줘"라고 요청 가능

### Q: Copilot의 답변이 마음에 안 들어요
**A**: 
- "더 자세히 설명해줘"
- "다른 방식으로 해줘"
- "예시를 추가해줘"
같은 추가 프롬프트를 입력하세요

### Q: 코딩을 전혀 모르는데 괜찮나요?
**A**: **완벽합니다!** 
- 이 워크숍은 비개발자를 위해 설계됨
- 모든 코드는 복사 & 붙여넣기
- 이해가 안 되면 언제든 강사에게 질문
