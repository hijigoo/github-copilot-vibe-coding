# Copilot 첫 사용하기
## ⏰ 소요 시간: 10분 (0:25 – 0:35)

---

## 🎯 이 세션에서 배울 내용

- Copilot Chat 열고 대화하기
- 첫 프롬프트 작성하기
- 간단한 표 만들어보기

---

## Step 1: Copilot Chat 열기 (1분)

**강사와 함께합니다**: 

1. 키보드에서 `Cmd + I` (Mac) 또는 `Ctrl + I` (Windows) 누르기
2. 화면에 Chat 창이 나타나면 성공!

**💡 Tip**: 왼쪽 사이드바에서 채팅 아이콘을 클릭해도 됩니다.

---

## Step 2: 첫 프롬프트 작성하기 (3분)

### 실습 1: 간단한 인사

**프롬프트**:
```
안녕하세요! 저는 비개발자입니다. GitHub Copilot이 무엇을 도와줄 수 있나요?
```

**기대 결과**:
- Copilot이 자신의 기능을 설명합니다
- 비개발자가 활용할 수 있는 방법을 제안합니다

### 실습 2: 간단한 문서 생성

**프롬프트**:
```
회의 참석자 5명의 이름과 역할을 표로 만들어줘. 
참석자는 김철수(PO), 이영희(UX), 박지민(마케터), 최민수(개발자), 정수진(디자이너)야.
```

**기대 결과**:
Copilot이 다음과 같은 표를 생성합니다:

| 이름 | 역할 |
|------|------|
| 김철수 | PO |
| 이영희 | UX |
| 박지민 | 마케터 |
| 최민수 | 개발자 |
| 정수진 | 디자이너 |

---

## 💡 Step 3: 좋은 프롬프트 작성 팁 (3분)

**구체적으로 말하기**
```
❌ "보고서 만들어줘"
✅ "지난주 프로젝트 진행 상황 보고서를 만들어줘. 
    완료된 작업, 진행 중인 작업, 다음 주 계획으로 구성해줘."
```

**형식 지정하기**
```
❌ "데이터 정리해줘"
✅ "이 데이터를 표 형식으로 정리해줘."
```

**핵심**: 명확하게 말할수록 원하는 결과가 나옵니다.

## ❓ 자주 묻는 질문

**Q: Copilot이 응답하지 않아요**
A: 강사에게 알려주세요. 함께 해결합니다.

**Q: 프롬프트를 잘못 입력했어요**
A: 다시 입력하거나 "다시 해줘"라고 요청하세요.

**Q: Copilot의 답변이 마음에 안 들어요**
A: "더 자세히 설명해줘" 또는 "다른 방식으로 해줘"라고 추가 요청하세요.
