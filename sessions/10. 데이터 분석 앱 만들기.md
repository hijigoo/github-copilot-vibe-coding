## 📖 시나리오

매주 새 CSV 파일이 도착할 때마다 엑셀 열고 정리하고 차트 만드는 일이 반복됩니다. 이번 세션에서는 **HTML 한 파일(`data_insight_app.html`)에서 버튼으로 CSV를 업로드하면 자동으로 요약·차트·표를 만들어 주는 앱**을 Copilot과 함께 만듭니다. 누가 만들어도 결과물은 동일한 하나의 HTML 파일이므로, 완성 후에는 파일만 전달하면 누구나 브라우저에서 분석 내용을 바로 확인할 수 있습니다.

**현재 문제:**
- ❌ CSV 구조가 달라질 때마다 새 파일/시트를 만들어야 함
- ❌ 차트/통계를 매번 수동으로 작성해야 해서 시간이 오래 걸림
- ❌ 분석 결과를 공유하려면 PPT나 PDF로 다시 정리해야 함

**Copilot으로 해결:**
- ✅ 버튼 하나로 CSV를 업로드하고 즉시 분석 결과를 보여주는 단일 HTML 앱 생성
- ✅ 자동 요약 카드, 차트, 상세 표를 한 화면에 표시
- ✅ `data_insight_app.html` 하나만 공유하면 누구나 결과 확인 가능

## 💬 실습: 데이터 분석 앱 만들기

Copilot에게 아래처럼 요청해주세요.

```
아래 내용에 대한 앱을 data_insight_app.html 파일로 만들어서 저장해줘.
카드와 차트는 그림자 효과가 있는 깔끔끔하고 전문적인 스타일로 디자인해줘.

브라우저 화면 중앙에 "CSV 파일 업로드" 버튼을 만들고, 버튼을 누르면 파일 선택 창이 뜨게 해줘.
CSV를 고르면 자동으로 내용을 읽어서, 다음 정보를 화면에 보여줘:

1) "데이터 요약" 그리드 형태: 행 수, 열 수, 숫자 열의 평균/최솟값/최댓값, 범주형 열의 TOP 5 항목 보여줘
2) "자동 차트" 영역: 숫자 열이 2개 이상이면 산점도, 1개뿐이면 히스토그램, 범주형 열이 있으면 막대그래프으로 보여줘
3) "상세 테이블" 영역: 전체 데이터를 테이블로 보여주되, 열 이름으로 정렬/검색할 수 있게 해줘

전체적으로 UI/UX 는 보기 편해야해. 
```

> **원하는 기능을 다시 설명해줄수록 Copilot이 더 정확한 결과를 제공합니다.** 예) "색상은 파랑 계열로", "차트는 반응형으로" 등
