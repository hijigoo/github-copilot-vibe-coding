# 📊 데이터 분석

## 🎯 이 섹션에서 배울 내용

GitHub Copilot을 활용하여 데이터를 분석하고 인사이트를 도출하는 방법을 배웁니다.
사용자 피드백 분석부터 데이터 시각화까지, PM과 기획자를 위한 실전 데이터 분석 기술을 익혀봅시다.

---

## 📚 세션 구성

### 1️⃣ 사용자 피드백 분석 (25분)

**무엇을 배우나요?**
- 정성적 피드백(인터뷰, 리뷰)을 구조화하여 분석
- 반복되는 패턴과 핵심 이슈 자동 추출
- UX 리서치 데이터 종합 분석
- 인사이트를 실행 가능한 액션 아이템으로 변환

**실무 활용 예시:**
- 고객 인터뷰 데이터 분석
- 앱스토어 리뷰 분석
- 사용자 설문조사 결과 정리
- 사용성 테스트 결과 분석

**소요 시간:** 25분  
**난이도:** ⭐⭐⭐☆☆

---

### 2️⃣ 데이터 기반 의사결정 (30분)

**무엇을 배우나요?**
- 정성 데이터와 정량 데이터 통합 분석 (트라이앵귤레이션)
- 데이터 기반 우선순위 결정 방법
- A/B 테스트 결과 분석
- 데이터로 설득력 있는 제안 만들기

**실무 활용 예시:**
- 기능 개선 우선순위 결정
- 리소스 배분 의사결정
- 제품 로드맵 수립
- 경영진 보고 자료 작성

**소요 시간:** 30분  
**난이도:** ⭐⭐⭐⭐☆

---

### 3️⃣ CSV 데이터를 HTML 리포트로 변환 (20분)

**무엇을 배우나요?**
- CSV 데이터를 시각화된 HTML 리포트로 변환
- 자동화 스크립트 생성
- 대시보드 형태의 리포트 만들기
- 주기적 리포트 자동화

**실무 활용 예시:**
- 주간/월간 지표 리포트 자동 생성
- 운영 데이터 시각화
- 마케팅 성과 리포트
- A/B 테스트 결과 공유

**소요 시간:** 20분  
**난이도:** ⭐⭐⭐☆☆

---

## 💡 이 섹션을 마치면

- ✅ 정성적 피드백을 체계적으로 분석할 수 있습니다
- ✅ 정성/정량 데이터를 통합하여 인사이트를 도출할 수 있습니다
- ✅ 데이터 기반으로 우선순위를 결정할 수 있습니다
- ✅ 자동화된 데이터 리포트를 생성할 수 있습니다
- ✅ 비개발자도 간단한 데이터 분석 스크립트를 만들 수 있습니다

## 🎓 학습 목표

1. **데이터 해석 능력**: 다양한 형태의 데이터에서 의미 있는 인사이트 발견
2. **의사결정 역량**: 데이터 기반으로 합리적인 결정 내리기
3. **자동화**: 반복적인 데이터 분석 작업 자동화
4. **커뮤니케이션**: 데이터를 시각화하여 효과적으로 전달

## 📊 왜 PM/기획자에게 데이터 분석이 중요한가요?

### 1. **객관적 의사결정**
```
❌ "제 생각에는 이 기능이 중요할 것 같아요"
✅ "사용자 인터뷰 결과 60%가 이 기능을 요청했고,
    데이터 분석 결과 전환율이 30% 개선될 것으로 예상됩니다"
```

### 2. **우선순위 설정**
```
💡 한정된 리소스로 최대의 임팩트!
   - 사용자 니즈 + 비즈니스 임팩트 + 구현 난이도
   - 데이터 기반 스코어링으로 우선순위 결정
```

### 3. **이해관계자 설득**
```
📊 데이터는 가장 강력한 설득 도구
   - 경영진: ROI, 비즈니스 임팩트
   - 개발팀: 사용자 데이터, 명확한 요구사항
   - 디자인팀: 사용성 테스트 결과
```
