# 1. 제품 아이디어 관리 도구 만들기

## ⏰ 소요 시간
**25분**

---

## 📖 시나리오

PO로서 제품 아이디어가 슬랙, 이메일, 메모장 곳곳에 흩어져 있습니다.
고객 피드백도 체계적으로 관리하기 어렵고, 우선순위 판단도 어렵습니다.

**현재 문제:**
- ❌ 아이디어가 산재되어 추적 어려움
- ❌ 피드백 관리가 체계적이지 않음
- ❌ 우선순위가 불명확
- ❌ 팀원 공유가 불편함

**만들 도구로 해결:**
- ✅ 한 곳에서 모든 아이디어 관리
- ✅ 카테고리별 분류 (아이디어/피드백/요구사항)
- ✅ 우선순위 명확화 (높음/중간/낮음)
- ✅ 파일로 저장/공유 가능

**💡 비개발자를 위한 설명:**
이번 세션에서는 Copilot이 웹 애플리케이션을 만들어줍니다!
여러분은 원하는 기능만 설명하면 됩니다.

---

## 🛠️ 실습: 제품 관리 도구 만들기

### Step 1: Copilot Chat 열기
Copilot Chat을 열고 아래와 같이 요청하세요.

### Step 2: Copilot에게 명령하기

```
PO를 위한 제품 아이디어 관리 도구를 만들어줘:

파일명: product_manager.html

기능:
1. 입력 폼:
   - 제목 (필수)
   - 카테고리 선택 (아이디어/피드백/요구사항)
   - 우선순위 선택 (높음/중간/낮음)
   - 상세 설명 (텍스트 영역)
   - 추가 날짜 (자동)

2. 목록 표시:
   - 카테고리별로 구분해서 표시
   - 우선순위에 따라 색상 다르게 (높음=빨강, 중간=주황, 낮음=초록)
   - 각 항목마다 수정/삭제 버튼

3. 파일 저장/불러오기:
   - "저장" 버튼: 모든 데이터를 TXT 파일로 다운로드
   - "불러오기" 버튼: TXT 파일을 선택해서 데이터 로드
   - 파일 형식: JSON 형태로 저장

4. 필터링:
   - 카테고리별 필터 버튼
   - 우선순위별 필터 버튼
   - "전체 보기" 버튼

디자인:
- 깔끔한 카드 스타일
- 보라색 계열 그라데이션 배경
- 반응형 (모바일에서도 예쁘게)

모든 코드를 하나의 HTML 파일에 작성하고,
바탕화면에 저장해줘.
```

**💡 Copilot Agent가 자동으로:**
1. ✅ 완전한 HTML 파일 생성
2. ✅ 파일 저장/불러오기 기능 구현
3. ✅ 바탕화면에 저장
4. ✅ 완료!

**여러분이 할 일**: 결과만 확인하면 됩니다!

---

## ✅ Step 3: 도구 실행하기 (2분)

1. 바탕화면에서 `product_manager.html` 파일 찾기
2. 파일을 **더블클릭**
3. 브라우저에서 도구가 열립니다!

**중요**: 서버 없이 바로 작동합니다!
