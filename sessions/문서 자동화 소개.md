# 문서 자동화 - 소개

## 💡 무엇을 배우나요?

30분 걸리던 회의록 정리를 5분으로, 2시간 걸리던 PRD 작성을 10분으로 단축하는 방법을 배웁니다.

---

## 📚 다룰 내용

**1. 회의록 자동 정리** (30분)
- 비정형 메모 → 구조화된 회의록
- Action Item 자동 추출
- 재사용 템플릿 생성

**2. 문서 자동 작성** (30분)
- 간단한 아이디어 → 완성된 PRD
- 문체 변환 (공식/비공식)
- PRD 템플릿 제작

---

## 🎯 이렇게 달라집니다

**Before**: 회의록 30분, PRD 작성 2시간  
**After**: 회의록 5분, PRD 작성 10분
