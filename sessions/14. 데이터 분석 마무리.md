# 11. 데이터 분석 마무리

## 🎉 완료!

세션 8~10을 통해 **텍스트 피드백 → 리포트 → 단일 HTML 앱**까지 완성했습니다. 이제 누구나 같은 데이터로 동일한 분석 결과를 재현할 수 있습니다.

---

## 📌 핵심 포인트

**Session 8 · 사용자 피드백 분석**
- 비정형 인터뷰 메모(`data/user_feedback_data.txt`)를 구조화하여 `user_feedback_table.md` 생성
- 카테고리·감정·우선순위를 한 번에 계산해 다음 단계 입력값으로 재사용

**Session 9 · 분석 보고서 제작**
- `user_feedback_table.md`를 참고해 `ux_research_report.html`과 `ux_research_report.xlsx`를 동시에 생성
- 불만사항 Top5 차트, 페르소나 카드, 개선 로드맵 타임라인으로 경영진/운영팀 요구 충족

**Session 10 · 데이터 분석 앱**
- CSV 업로드만으로 통계·차트·테이블을 뽑아주는 `data_insight_app.html`을 완성
- 누구에게든 HTML 파일 하나만 전달하면 브라우저에서 분석 결과 확인 가능

---

## 📁 확보된 산출물
- `user_feedback_table.md`: 카테고리별 인사이트, 페르소나 그룹, 감정·우선순위 지표
- `ux_research_report.html / .xlsx`: 차트·타임라인·다운로드용 시각 리포트 쌍
- `data_insight_app.html`: CSV 업로드 버튼, 자동 요약 카드, 반응형 차트/테이블을 담은 단일 파일 앱

---

## 🧪 프롬프트 패턴
```
1) "data/user_feedback_data.txt를 구조화해서 [필드 목록]을 가진 Markdown 테이블로 정리해줘"
2) "user_feedback_table.md 내용을 참고해 HTML과 Excel 리포트를 각각 저장해줘"
3) "CSV 업로드 버튼이 있는 data_insight_app.html을 만들고, 업로드 후 요약/차트/테이블을 한 화면에 보여줘"
```
필요한 산출물·스타일·파일명을 구체적으로 명시하면 반복 작업 없이 동일 결과를 재생산할 수 있습니다.
