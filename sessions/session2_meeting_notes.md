# Session 2: 회의록 자동 정리
## ⏰ 소요 시간: 30분 (0:35 – 1:05)

---

## 🎯 학습 목표

이 세션을 마치면:
- 회의록에서 Action Item 자동 추출
- 담당자, 마감일, 작업 내용을 표로 정리
- 재사용 가능한 회의록 템플릿 생성

---

## 📖 실무 시나리오

**상황**: 주간 제품 회의를 마쳤습니다. 30분간 논의한 내용을 메모했지만 정리되지 않은 상태입니다. 
팀원들에게 공유할 회의록을 만들어야 합니다. 특히 **누가, 언제까지, 무엇을** 해야 하는지 명확히 정리해야 합니다.

**문제점**
- 회의록 정리에 30분 이상 소요
- Action Item을 빠뜨리는 경우 발생
- 담당자와 마감일이 불명확

**목표**
- Copilot으로 5분 내 정리 완료
- 모든 Action Item 자동 추출
- 명확한 표 형식으로 정리

결정 사항:
- 리뷰 섹션을 상단으로 이동 (이번 주 배포)
- 이미지 로딩 속도 개선 (다음 주 완료)
- 필터 UI 개선 (다음 스프린트)

다음 회의: 11월 20일 (수) 오전 10시

## 💪 실습 1: Action Item 자동 추출하기 (10분)

Copilot Agent에게 자연어로 요청하면 자동으로 처리됩니다.

**Step 1: Copilot Chat 열기**
`Cmd + Shift + I` (Mac) / `Ctrl + Shift + I` (Windows)

### Step 2: Copilot에게 명령하기

Chat 창에 다음과 같이 요청하세요:

```
sessions/data/sample_meeting_notes.txt 파일을 읽어서 
다음 정보를 추출해서 보여줘:

1. 회의 날짜와 주제
2. 참석자 목록 (이름과 역할)
3. Action Items (작업 내용, 담당자, 마감일)
4. 주요 결정 사항

결과는 Markdown 표 형식으로 보여줘.
```

**Step 3: Copilot Agent가 자동 실행**

Copilot Agent가 자동으로:
- 코드 생성
- 파일 읽기
- 결과 정리

여러분은 결과만 확인하면 됩니다.

### Step 4: 결과 확인 및 수정 요청

만족스럽지 않으면 자연어로 다시 요청:

```
마감일을 더 구체적으로 추출해줘
```

```
우선순위를 높음/중간/낮음으로 자동 판단해줘
```

```
이 결과를 HTML 리포트로 만들어서 파일로 저장해줘
```

### 💡 Copilot 활용 팁

**더 정교한 요청 예시**:

```
위 회의록에서 담당자별로 작업을 그룹화해서 보여줘
```

```
마감일이 이번 주인 항목만 필터링해줘
```

```
이 Action Items를 간트 차트(Mermaid)로 시각화해줘
```

**파일 저장이 필요한 경우**:
```
위 결과를 meeting_summary.md 파일로 저장해줘
```

Copilot이 자동으로 파일을 생성하고 저장합니다.

## 🎯 예상 결과물 예시

```markdown
# 숙박 상품 상세 페이지 개선 회의 - Action Items

**회의 날짜**: 2024년 11월 13일  
**다음 회의**: 2024년 11월 20일 (수) 오전 10시

---

## 📋 Action Items

| 우선순위 | 작업 내용 | 담당자 | 마감일 | 상태 |
|---------|-----------|--------|--------|------|
| 높음 | 리뷰 섹션 위치 변경 와이어프레임 제작 | 이지혜 | 11월 14일 (목) | 진행 전 |
| 높음 | 리뷰 섹션 위치 변경 프론트 개발 | 최수진 | 11월 17일 (금) | 진행 전 |
| 높음 | 이미지 로딩 속도 개선 (CDN 설정) | 박준형 | 11월 20일 (수) | 진행 전 |
| 중간 | A/B 테스트 시나리오 및 성과 측정 기준 정리 | 정다혜 | 11월 20일 (월) | 진행 전 |
| 낮음 | 옵션 필터 UI 개선 (토글 방식) | 이지혜, 최수진 | 다음 스프린트 | 진행 전 |

---

## ✅ 주요 결정 사항

1. **우선순위 설정**:
   - 1순위: 리뷰 섹션 이동 (이번 주 배포 목표)
   - 2순위: 이미지 최적화 (다음 주 완료)
   - 3순위: 필터 개선 (다음 스프린트 반영)

2. **목표 지표**:
   - 현재 이탈률: 35%
   - 목표 이탈률: 22% (경쟁사 수준)

3. **배포 일정**:
   - 리뷰 섹션 변경: 11월 17일 (금)
   - A/B 테스트 시작: 배포 직후
```

## 💪 실습 2: 고급 분석 및 시각화 (10분)

이번에는 더 멋진 결과를 만들어봅시다!

### 🎯 목표
담당자별 작업 현황을 시각화하고 HTML 리포트 만들기

### Copilot에게 명령하기

```
위에서 추출한 Action Items를 다음과 같이 분석해줘:

1. 담당자별로 작업 그룹화
2. 우선순위별 통계 (높음/중간/낮음 개수)
3. 이번 주 마감인 항목만 필터링
4. HTML 리포트로 만들어서 파일로 저장

HTML에는 다음을 포함:
- 담당자별 작업 목록 표
- 우선순위 파이 차트
- 마감일 타임라인
- 스타일링 (그라데이션 배경, 카드 디자인)
```

### 강사와 함께 실행하기

Copilot이 생성한 코드를 **강사의 안내**에 따라 실행합니다:

1. **강사**: "생성된 코드를 복사하세요"
2. **강사**: "새 파일을 만들고 붙여넣으세요"
3. **강사**: "터미널을 열어서 이 명령어를 입력하세요"
4. **결과**: HTML 파일이 생성되고 브라우저에서 열립니다!

**💡 Tip**: 처음에는 어려워 보이지만, 2-3번 반복하면 익숙해집니다!

### 추가 요청 예시

**담당자에게 보낼 이메일 초안**:
```
각 담당자에게 보낼 작업 알림 이메일을 작성해줘.
이름과 담당 작업 목록을 포함해서 친근한 톤으로.
```

**간트 차트로 시각화**:
```
이 Action Items를 Mermaid 간트 차트로 만들어줘.
담당자별로 색상을 다르게 해줘.
```

**Slack 메시지 형식**:
```
이 Action Items를 Slack에 공유할 메시지 형식으로 만들어줘.
이모지와 포맷팅을 포함해서.
```

## 💪 실습 3: 회의록 템플릿 자동 생성 (10분)

마지막으로, 재사용 가능한 템플릿을 만들어봅시다.

### Copilot에게 명령하기

```
제품 개발팀의 주간 회의록 템플릿을 만들어서 
meeting_template.md 파일로 저장해줘.

다음 섹션을 포함:
1. 회의 기본 정보 (날짜, 시간, 참석자)
2. 지난 회의 Action Items 리뷰
3. 주요 논의 사항
4. 결정 사항
5. 새로운 Action Items (표 형식)
6. 다음 회의 안건
7. 참고 자료 링크

각 섹션마다 작성 가이드를 주석으로 추가하고,
이모지로 시각적으로 보기 좋게 만들어줘.
```

**완료!** Copilot이 자동으로 파일을 생성합니다.

## 💡 실무 활용 팁

### 1. 회의 중 실시간 활용
- 회의록을 메모장에 간단히 작성
- 회의 끝나자마자 Copilot으로 정리 (5분 소요)
- 정리된 내용을 즉시 참석자에게 공유

### 2. 정기 회의 자동화
- 템플릿을 팀 공유 폴더에 저장
- 매주 같은 형식으로 작성
- 지난 회의록과 비교하며 진행 상황 추적

### 3. 다른 팀과 협업
- 일관된 형식으로 회의록 공유
- Action Item을 Jira/Asana 등 프로젝트 관리 도구로 복사
- 마감일 자동 알림 설정

## 🎓 추가 도전 과제

시간이 남았다면 다음을 시도해보세요:

### 도전 1: 회의록 요약본 만들기
```
위의 회의록을 경영진 보고용으로 3줄 요약해줘.
각 줄은 50자 이내로 작성해줘.
```

### 도전 2: 회의록을 이메일 형식으로 변환
```
위의 회의록을 팀원들에게 보낼 이메일 형식으로 변환해줘.
제목과 본문을 포함하고, 친근한 톤으로 작성해줘.
```

### 도전 3: Mermaid 다이어그램으로 시각화
```
이 Action Items를 Mermaid 간트 차트로 시각화해줘.
```

## 🔍 자주 묻는 질문

### Q1: 회의록이 너무 길면 어떻게 하나요?
**A**: 회의록을 섹션별로 나누어 각각 요청하세요.
```
위 회의록의 "논의 사항" 부분만 정리해줘.
```

**A**: Copilot에게 추론을 요청하세요.
```
위 회의록에서 각 작업의 적절한 담당자를 제안해줘.
단, 역할 기반으로 추천해줘.
```

**A**: 네! 각 회의록을 구분하고 일괄 정리를 요청하세요.

아래 3개의 회의록에서 모든 Action Items를 하나의 표로 통합해줘.
