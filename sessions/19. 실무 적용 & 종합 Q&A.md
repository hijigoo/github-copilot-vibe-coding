### 섹션별 핵심 학습

**📝 문서 자동화**
- 회의록 30분 → 5분으로 단축
- 재사용 가능한 템플릿 앱 제작
- Action Item 자동 추출 및 HTML 리포트 생성

**📊 데이터 분석**
- 정성 데이터 자동 분류 및 패턴 발견
- 페르소나 카드, 개선 로드맵 타임라인 생성
- CSV 업로드만으로 통계·차트·테이블 자동 생성

**🛠️ 실전 도구 만들기**
- 비개발자도 Copilot으로 웹 애플리케이션 제작
- TXT(JSON) 저장/불러오기, 필터링 기능 구현
- 실무에서 바로 배포 가능한 도구 완성

## 🧪 프롬프트 패턴 정리

### 문서 자동화 패턴
```
1) "data/sample_meeting_notes.txt를 읽고 Action Item/담당자/마감일을 Markdown 표로 정리해줘"
2) "위 표를 바탕으로 meeting_action_items.html이라는 시각화 리포트를 저장해줘"
3) "작성자가 섹션별로 입력하고 txt로 저장할 수 있는 meeting_template_app.html을 만들어줘"
```

### 데이터 분석 패턴
```
1) "data/user_feedback_data.txt를 구조화해서 [필드 목록]을 가진 Markdown 테이블로 정리해줘"
2) "user_feedback_table.md 내용을 참고해 HTML과 Excel 리포트를 각각 저장해줘"
3) "CSV 업로드 버튼이 있는 data_insight_app.html을 만들고, 업로드 후 요약/차트/테이블을 한 화면에 보여줘"
```

### 실전 도구 패턴
```
1) "[사용자/팀]이 사용할 [도구명].html을 만들고, 아래 입력 필드와 동작을 모두 포함해줘"
2) "데이터를 TXT(JSON)로 저장/불러오기 기능을 추가하고, 버튼/토스트/필터 UI를 넣어줘"
3) "테스트 시나리오를 미리 채워둔 버튼을 추가하고, 클릭 시 입력값이 자동으로 기입되게 해줘"
```
