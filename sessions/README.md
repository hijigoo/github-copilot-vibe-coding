# 👋 워크숍 소개

## 🎯 워크숍 개요

이 워크숍은 **비개발자(Product Owner, 마케터)**를 위한 GitHub Copilot 실전 활용 교육입니다. 
코딩 지식 없이도 AI의 도움으로 업무 자동화 도구를 만들 수 있습니다.

## 👥 참가 대상

- 16명의 Product Owner
- 1명의 마케터
- **코딩 경험이 없어도 괜찮습니다!**

## ⏱️ 총 소요 시간

**3시간 50분** (휴식 시간 포함)

## 📚 커리큘럼

### 🚀 시작하기 (35분)
1. **환경 세팅** (25분) - VS Code와 GitHub Copilot 설치
2. **GitHub Copilot 첫 사용하기** (10분) - 기본 사용법 익히기

### 📝 문서 자동화 (60분)
1. **회의록 자동 정리** (30분) - 회의록에서 액션 아이템 자동 추출
2. **문서 자동 작성** (30분) - PRD(Product Requirements Document) 자동 작성

### 📊 데이터 분석 (75분)
1. **사용자 피드백 분석** (15분) - 피드백 분류 및 페르소나 분석
2. **데이터 기반 의사결정** (15분) - 정량 데이터 통합 및 액션 아이템 생성
3. **CSV 데이터를 HTML 리포트로 변환** (30분) - 데이터 시각화 리포트 생성

### 🛠️ 실전 도구 만들기 (40분)
1. **제품 아이디어 관리 도구 만들기** (40분) - 웹 기반 관리 도구 개발

### 🎓 마무리 (20분)
1. **실무 적용 & 종합 Q&A** (20분) - 실무 활용법과 질의응답

## 💡 이 워크숍에서 배우는 것

- ✅ GitHub Copilot을 활용한 업무 자동화
- ✅ 코드 없이 Python 스크립트 만들기
- ✅ 데이터 분석 및 시각화
- ✅ 웹 애플리케이션 개발
- ✅ 실무에 바로 적용 가능한 실전 기술

## 🎁 워크숍 후 얻게 되는 것

1. **업무 효율 향상**: 반복 작업을 자동화하여 시간 절약
2. **데이터 활용 능력**: 엑셀이 아닌 프로그래밍으로 데이터 분석
3. **개발자와의 소통**: 기술적 대화가 가능한 이해도 향상
4. **실전 도구**: 바로 사용 가능한 9개의 실습 프로젝트
