# 👋 워크숍 소개

## 🎯 워크숍 개요

이 워크숍은 **현업 전문가(Product Owner, 마케터, 기획자)**&#xB97C; 위한 GitHub Copilot 실전 활용 교육입니다. 코딩 지식 없이도 AI의 도움으로 업무 자동화 도구를 만들 수 있습니다.

## 👥 참가 대상

* **코딩 경험이 없어도 괜찮습니다!**

## ⏱️ 총 소요 시간

**약 3시간** (휴식 시간 포함)

## 💡 이 워크숍에서 배우는 것

* ✅ GitHub Copilot을 활용한 업무 자동화
* ✅ 코드 없이 Python 스크립트 만들기
* ✅ 데이터 분석 및 HTML 시각화 리포트 생성
* ✅ 웹 애플리케이션 직접 제작
* ✅ 실무에 바로 적용 가능한 프롬프트 패턴

## 🎁 워크숍 후 얻게 되는 것

1. **업무 효율 향상**: 반복 작업을 자동화하여 시간 절약
2. **데이터 활용 능력**: 엑셀이 아닌 프로그래밍으로 데이터 분석
3. **개발자와의 소통**: 기술적 대화가 가능한 이해도 향상
4. **실전 도구**: 바로 사용 가능한 웹앱과 리포트
