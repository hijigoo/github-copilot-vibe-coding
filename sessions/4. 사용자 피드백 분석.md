# 1. 사용자 피드백 분석

## ⏰ 소요 시간
**25분**

---

## 📖 시나리오

UX 리서처로서 최근 신규 숙박 검색 기능에 대해 15명의 사용자를 인터뷰했습니다.
각 인터뷰 내용을 정리했지만, 이제 이 데이터를 분석하여 공통 패턴을 찾고 인사이트를 도출해야 합니다.

**현재 문제:**
- ❌ 피드백 데이터가 많아 분석에 시간 소요
- ❌ 수작업 분류 시 주관적 판단 개입
- ❌ 중요한 인사이트를 놓칠 가능성

**Copilot으로 해결:**
- ✅ 객관적이고 체계적인 분석
- ✅ 페르소나별 니즈 자동 분류
- ✅ 우선순위 높은 이슈 자동 식별

## 💬 실습 1: 피드백 자동 분류 및 키워드 추출

### 🎯 목표
Copilot에게 명령만 하면 **자동으로 분석되고 결과가 나옵니다**!

### Step 1: Copilot Chat 열기
```
⌘ + Shift + I (macOS) / Ctrl + Shift + I (Windows)
```

### Step 2: Copilot에게 명령하기

Chat 창에 다음과 같이 요청하세요:

```
data/user_feedback_data.txt 파일을 분석해줘:

1. 가장 많이 언급된 불만사항 Top 5
2. 각 불만사항별 언급 횟수
3. 불만사항을 카테고리별로 분류 (성능, UI/UX, 기능, 가격, 콘텐츠)
4. 각 카테고리별 주요 키워드
5. 페르소나별 피드백 그룹화 (연령대, 직업)

결과를 Markdown 표 형식으로 보여줘.
```

### Step 3: 코드 실행하기

**진행 순서**:
1. Copilot이 Python 코드를 생성합니다
2. 생성된 코드를 새 파일에 저장합니다
3. 실행 버튼을 눌러서 코드를 실행합니다
4. 결과를 확인합니다

**💡 Tip**: Copilot Agent가 자동으로 코드를 실행해주는 경우도 있습니다!

### Step 4: 추가 분석 요청

만족스럽지 않으면 계속 요청:

```
감정 분석도 추가해줘 (긍정/부정/중립)
```

```
심각도를 높음/중간/낮음으로 자동 판단해줘
```

```
개선 우선순위를 1~10점으로 계산해줘
```

**HTML 리포트로 변환하기**:
```
위 UX 리서치 분석 결과를 인터랙티브한 HTML 리포트로 만들어서
ux_research_report.html 파일로 저장해줘.

다음 내용 포함:
- 불만사항 Top 5 (차트)
- 카테고리별 분석 (접을 수 있는 섹션)
- 페르소나별 인사이트 (카드 레이아웃)
- 개선 로드맵 타임라인
- 반응형 디자인 (모바일 지원)
- 인쇄 가능한 스타일
```

### 💡 Copilot 활용 팁

**더 정교한 요청 예시**:

```
20대 사용자의 불만사항만 필터링해서 보여줘
```

```
"성능" 카테고리의 불만만 상세하게 분석해줘
```

```
각 페르소나의 주요 니즈를 3줄로 요약해줘
```

**시각화 요청**:
```
불만사항 키워드를 Word Cloud로 시각화해서 이미지로 저장해줘
```

```
카테고리별 분포를 파이 차트로 만들어줘
```

## 🎓 추가 도전 과제

시간이 남았다면 다음을 시도해보세요:

### 도전 1: 감정 분석 추가
```
각 피드백의 감정을 분석해줘 (긍정/부정/중립).
감정별로 통계를 내고, 가장 부정적인 이슈 TOP 5를 뽑아줘.
```

### 도전 2: 워드 클라우드 데이터 생성
```
피드백에서 자주 언급된 단어의 빈도수를 계산해줘.
워드 클라우드 생성을 위한 JSON 형식으로 출력해줘.
```

### 도전 3: 경쟁사 비교 분석
```
피드백에서 언급된 경쟁사(야놀자, 에어비앤비 등)의 
기능을 정리하고, 우리 서비스와 비교 분석해줘.
```
