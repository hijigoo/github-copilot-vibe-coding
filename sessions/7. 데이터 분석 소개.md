# 1. 데이터 분석 소개

## 💡 무엇을 배우나요?

사용자 피드백부터 CSV 데이터까지, 데이터를 분석하고 실행 가능한 인사이트로 만드는 방법을 배웁니다.

***

## 📚 다룰 내용

**1. 사용자 피드백 분석** (25분)

* 정성 데이터 구조화
* 패턴 & 이슈 자동 추출
* 액션 아이템 도출

**2. 데이터 기반 의사결정** (30분)

* 정성/정량 데이터 통합 (트라이앵귤레이션)
* 데이터 기반 우선순위 결정
* 설득력 있는 제안 만들기

**3. CSV → HTML 리포트** (20분)

* 데이터 시각화
* 자동화 스크립트
* 주기적 리포트 생성
