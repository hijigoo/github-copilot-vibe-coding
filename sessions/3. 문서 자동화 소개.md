# 문서 자동화 - 소개

## 📖 시나리오

주간 회의가 끝나면 누구보다 먼저 공유 문서를 작성해야 하는 PO가 있습니다. 
노션에 흩어진 메모, 슬랙 스레드, 음성 메모를 모아 회의록을 만들고 Action Item 정리까지 하려면 최소 30분이 걸립니다. 
그사이 팀원들은 다음 업무로 넘어가 버리고, 정리가 늦어지면 담당자와 마감일도 흐릿해집니다.

**현재 문제:**
- ❌ 회의 메모마다 형식이 달라 복붙과 서식 정리에 시간이 소요
- ❌ Action Item 담당자·마감일을 빠뜨려 follow-up이 지연
- ❌ 공유용 파일/HTML을 만드는 과정이 수동이라 속도가 나지 않음

**Copilot으로 해결:**
- ✅ 비정형 메모를 1분 만에 구조화된 회의록으로 변환
- ✅ 담당자·우선순위·마감일을 자동 추출해 표 형태로 정리
- ✅ 재사용 가능한 회의록 작성 앱을 한 번에 생성하고 txt로 내보내기까지 완료

---

## 📚 다룰 내용

**1. 회의록 자동 정리 (Session 4)**
- `data/sample_meeting_notes.txt`에서 Action Item, 결정 사항을 자동 추출

- 담당자별 통계, 시각화, Slack/E-mail용 포맷 생성

**2. 회의록 작성 앱 만들기 (Session 5)**
- 재사용 가능한 회의록 템플릿 앱(`meeting_template_app.html`) 생성
- 필수 섹션(기본 정보, 논의 사항, 결정 사항, Action Item) 자동 배치
- 작성 즉시 txt 파일로 내보내어 슬랙/이메일에 바로 공유
