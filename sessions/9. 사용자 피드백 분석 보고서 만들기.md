## 📖 시나리오

전 세션에서 `user_feedback_table.md`를 완성했지만, 경영진 리뷰를 위해서는 시각화된 HTML 리포트와 엑셀 파일이 필요합니다. 특히 "불만사항 Top 5 차트"와 "개선 로드맵 타임라인"을 한눈에 보여줘야 하고, 운영팀은 동일한 데이터를 엑셀로 내려받아 추가 가공을 하길 원합니다. Copilot에게 동일한 데이터 소스를 제공하고, 한 번의 지시로 두 가지 형태의 리포트를 생성합니다.

**현재 문제:**
- ❌ Markdown 분석 결과만으로는 경영진에게 임팩트 있게 공유하기 어려움
- ❌ HTML·엑셀 버전을 따로 만들면 중복 작업과 포맷 불일치 발생
- ❌ 차트/타임라인/페르소나 카드 등 시각 요소를 직접 만들 시간이 부족

**Copilot으로 해결:**
- ✅ `user_feedback_table.md` 내용을 참조해 `ux_research_report.html`을 생성 (차트, 카드, 타임라인 포함)
- ✅ 동일 데이터를 기반으로 `ux_research_report.xlsx`를 만들어 운영팀이 바로 열 수 있게 제공
- ✅ 반응형 디자인·인쇄 모드·엑셀 시트 구성 등 반복 작업을 자연어 프롬프트로 정의

## 🎓 추가 도전 과제

시간이 남았다면 다음을 시도해보세요:

### 도전 1: 브랜드 테마 적용
```
ux_research_report.html에 브랜드 컬러, 로고, 폰트를 반영한 테마를 적용해줘.
Dark/Light 모드를 전환할 수 있는 토글도 추가해줘.
```

### 도전 2: 자동 이메일 요약 생성
```
HTML 리포트 핵심 내용을 5줄 요약으로 정리해줘.
경영진에게 보낼 이메일 형식(제목+본문)으로 작성하고, 리포트 링크/첨부 안내 문구를 포함해줘.
```

### 도전 3: 엑셀 피벗 시트 추가
```
ux_research_report.xlsx에 피벗 시트를 하나 더 만들어줘.
행=카테고리, 열=감정(부정/중립/긍정), 값=피드백 수 로 구성해줘.
```
