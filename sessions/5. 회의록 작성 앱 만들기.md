# 5. 회의록 작성 앱 만들기

## ⏰ 소요 시간
**30분**

---

## 📖 시나리오

제품팀 주간 회의를 마치면 항상 슬랙에 공유할 회의록을 급하게 정리해야 합니다. 하지만 매번 문서를 새로 만들다 보니 형식이 들쭉날쭉하고, 담당자·마감일을 빠르게 입력하기도 어렵습니다. 한 번 정리한 회의록을 재사용하려 해도 복사/붙여넣기 과정에서 서식이 깨져 다시 손이 많이 갑니다.

**현재 문제:**
- ❌ 회의록 템플릿이 없어 매번 처음부터 작성
- ❌ 담당자/마감일 입력 누락으로 후속 조치가 지연
- ❌ txt/markdown 저장 프로세스가 없어 공유용 파일 만들기가 번거로움

**Copilot으로 해결:**
- ✅ 재사용 가능한 회의록 템플릿 앱을 한 번에 생성
- ✅ 모든 필수 섹션(기본 정보, Action Item, 결정 사항)을 자동 배치
- ✅ 작성 즉시 txt 파일로 내보내 공유까지 바로 완료

---



## 👩🏻‍💻 실습 1: 회의록 템플릿이 있는 앱 만들기

마지막으로, 재사용 가능한 템플릿 앱을 만들어봅시다.

### Copilot에게 명령하기

```
제품 개발팀의 주간 회의록 템플릿을 만들어서
meeting_template_app.html 파일로 저장해줘.
사용자가 각 칸을 모두 작성한다음에 저장 버튼을 누르면 txt 파일로 저장할 수 있게 해줘.

다음 섹션을 포함:
1. 회의 기본 정보 (날짜, 시간, 참석자)
2. 지난 회의 Action Items 리뷰
3. 주요 논의 사항
4. 결정 사항
5. 새로운 Action Items (표 형식)
6. 다음 회의 안건
7. 참고 자료 링크

각 섹션마다 작성 가이드를 주석으로 추가하고,
이모지로 시각적으로 보기 좋게 만들어줘.
```

**완료!** Copilot이 자동으로 파일을 생성합니다.
