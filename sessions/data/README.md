# 실습 데이터 파일 설명

이 폴더에는 각 세션에서 사용할 실습용 샘플 데이터가 포함되어 있습니다.

## 📁 파일 목록

### `sample_meeting_notes.txt`
- **용도**: 회의록 자동 정리 실습 (Session 2)
- **내용**: 실제 회의록 형식의 텍스트
- **크기**: 6개 회의, 17개 Action Items

### `user_feedback_data.txt`
- **용도**: 사용자 피드백 분석 실습 (Session 4)
- **내용**: 30명의 사용자 피드백 (연령대, 직업, 의견 포함)
- **특징**: 긍정/부정/중립 의견이 섞여있음

### `customer_satisfaction_survey.csv`
- **용도**: CSV 데이터 분석 및 리포트 생성 (Session 5)
- **내용**: 고객 만족도 설문조사 결과 (50건)
- **컬럼**: 응답ID, 연령대, 직업, 만족도 점수(5개), 불만사항, 추천도
- **인코딩**: UTF-8

### `customer_list_full.csv`
- **용도**: 업무 자동화 실습 (Session 7)
- **내용**: 고객 정보 및 예약 상태 (20건)
- **컬럼**: 고객ID, 이름, 이메일, 전화번호, 상태, 등록일, 금액, 지역
- **인코딩**: UTF-8

### `email_extraction_sample.txt`
- **용도**: 정규표현식 실습 (Session 7)
- **내용**: 이메일 주소가 포함된 텍스트
- **특징**: 다양한 형식의 이메일 주소 포함

### `feature_concept.txt`
- **용도**: PRD/문서 자동 작성 실습 (Session 5)
- **내용**: "스마트 가격 알림" 기능 기획 배경, 핵심 기능, 목표 지표 정리
- **특징**: 그대로 붙여서 프롬프트 입력에 활용 가능, 한글 UTF-8 텍스트

## 📊 데이터 특징

- 실제 업무에서 사용할 법한 현실적인 데이터
- 적절한 크기 (실습 시간 내 처리 가능)
- 한글/영문 혼용
- UTF-8 인코딩 (한글 깨짐 방지)
