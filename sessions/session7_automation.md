## 🎯 학습 목표

이 세션을 마치면:
- Excel 파일을 자동으로 필터링하고 정리할 수 있습니다
- 반복되는 데이터 작업을 자동화할 수 있습니다
- 실제 업무에 바로 적용 가능한 스크립트를 만들 수 있습니다

---

## 📖 시나리오

> **상황**: PO인 여러분은 매주 영업팀으로부터 고객 리스트 Excel 파일을 받습니다. 이 파일에서 "결제 완료" 고객만 추출하여 새 파일로 저장하는 작업을 매주 반복하고 있습니다.

**문제점**:
- 매주 같은 필터링 작업 반복 (30분 소요)
- 수작업으로 인한 실수 발생 (행 누락, 잘못된 복사)
- 단순 반복 작업에 시간 낭비

**목표**:
- Copilot으로 10분 만에 자동화 스크립트 완성
- 앞으로는 클릭 한 번으로 작업 완료

## 💪 실습: Excel 자동 필터링 스크립트 만들기 (25분)

### 📝 시나리오 상세

매주 영업팀에서 고객 리스트 Excel 파일을 받습니다. 이 파일에서:
- "결제 완료" 상태인 고객만 필터링
- 필요한 컬럼만 추출 (이름, 이메일, 전화번호, 금액)
- 오늘 날짜가 포함된 새 파일로 저장
- 결과 요약 출력 (전체/완료 고객 수, 총 금액)

---

### Step 1: 샘플 데이터 확인

**샘플 파일 위치**: `sessions/data/customer_list_full.csv`

**데이터 구조**:
```
고객ID, 이름, 이메일, 전화번호, 상태, 등록일, 금액, 지역
1001, 김민수, minsu@email.com, 010-1234-5678, 결제 완료, 2024-11-01, 150000, 서울
1002, 이지은, jieun@email.com, 010-2345-6789, 대기 중, 2024-11-02, 200000, 부산
...
```

---

### Step 2: Copilot Chat 열기

```
⌘ + Shift + I (macOS) / Ctrl + Shift + I (Windows)
```

---

### Step 3: Copilot에게 명령하기

```
sessions/data/customer_list_full.csv 파일을 읽어서 다음 작업을 수행해줘:

1. "상태" 컬럼이 "결제 완료"인 행만 필터링
2. 필요한 컬럼만 선택: 이름, 이메일, 전화번호, 금액
3. 새 파일로 저장: "completed_customers_YYYY-MM-DD.csv"
4. 완료 후 다음 정보 출력:
   - 전체 고객 수
   - 결제 완료 고객 수
   - 총 결제 금액
   - 평균 결제 금액

엑셀 처리 도구(pandas) 사용.
오류 처리 포함.
모든 과정을 자동으로 실행해줘.
```

---

### Step 4: Copilot이 자동으로 실행

**💡 여러분은 아무것도 안 하셔도 됩니다!**

Copilot Agent가 자동으로:
1. ✅ Python 코드 생성
2. ✅ 필요한 패키지 자동 설치
3. ✅ 코드 자동 실행
4. ✅ 결과 파일 생성
5. ✅ 결과 출력

**여러분이 할 일**: 결과 확인만!

---

### 📊 예상 결과

```
✅ 파일을 읽었습니다. 전체 고객 수: 50명
📊 결제 완료 고객 수: 32명
💰 총 결제 금액: 5,840,000원
✅ 저장 완료: completed_customers_2024-11-13.csv

==================================================
📋 작업 요약
==================================================
전체 고객:           50명
결제 완료 고객:      32명 (64.0%)
총 결제 금액:        5,840,000원
평균 결제 금액:      182,500원
==================================================
```

**생성된 파일**: `completed_customers_2024-11-13.csv`

## 🔍 자주 묻는 질문

### Q1: 회사 데이터로 바로 써도 되나요?
**A**: 네! 다만 처음엔 복사본으로 테스트하세요. 원본 파일은 항상 백업하세요.

### Q2: Excel 파일 형식이 다르면?
**A**: Copilot에게 실제 파일 구조를 보여주고 수정 요청하세요:
```
내 파일은 컬럼 이름이 다릅니다:
- "상태" 대신 "결제상태"
- "이름" 대신 "고객명"
코드를 이에 맞게 수정해줘.
```

### Q3: 에러가 나면?
**A**: 에러 메시지를 복사해서 Copilot에게 보여주세요:
```
다음 에러가 발생했어:
[에러 메시지 붙여넣기]

어떻게 해결하면 될까?
```
