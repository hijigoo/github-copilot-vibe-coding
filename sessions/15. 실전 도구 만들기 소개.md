# 1. 실전 도구 만들기 소개

## 💡 무엇을 배우나요?

현업 전문가도 45분 만에 실제 동작하는 웹 애플리케이션 2개를 만듭니다!
요구사항만 명확히 말하면 Copilot이 CRUD, 계산 로직, 저장 기능까지 포함된 실전 도구를 만들어줍니다.

---

## 📚 다룰 내용

### 2. 제품 아이디어 관리 도구 (25분)

PO로서 흩어진 아이디어를 한 곳에서 관리하는 도구를 만듭니다.

- 아이디어/피드백/요구사항 입력 폼
- 카테고리별 분류 (아이디어/피드백/요구사항)
- 우선순위 색상 구분 (높음=빨강, 중간=주황, 낮음=초록)
- TXT(JSON) 저장/불러오기 기능
- 수정/삭제 버튼, 필터링 기능

### 3. 예약·수수료 계산기 (20분)

숙박 파트너와 프로모션 팀을 위한 정산 계산기를 만듭니다.

- 1박 요금, 숙박일수, 쿠폰, 수수료(%) 입력
- 고객 최종 결제 금액 · 플랫폼 수수료 · 업주 수령액 자동 계산
- 쿠폰 과다 적용 시 경고 토스트
- 예시 시나리오 자동 입력 버튼
- 결과 클립보드 복사 기능

## 🎯 이렇게 달라집니다

| Before | After |
|--------|-------|
| 아이디어가 슬랙, 이메일, 메모장에 흩어짐 | 한 곳에서 체계적으로 관리 |
| 요금 계산 시 수작업으로 실수 발생 | 입력만 하면 즉시 정확한 정산 |
| 도구 하나 만드는 데 개발자 요청 필요 | 프롬프트만으로 직접 제작 |

## 📚 다음 세션 미리보기

**🛠️ 실전 도구 만들기 > 2. 제품 아이디어 관리 도구 만들기**에서는:

* 아이디어/피드백/요구사항을 한 화면에서 입력·분류·필터링
* TXT(JSON) 저장/불러오기 기능 구현
* 우선순위 색상, 카드 UI를 Copilot 프롬프트만으로 구현

**소요 시간:** 25분
**준비물:** 없음 (Copilot만 있으면 OK!)
